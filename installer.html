IT ServiceDesk — Auto Installer

ติดตั้ง Google Sheets Database อัตโนมัติ ใช้เวลา ~2 นาที ไม่ต้องเขียน Code

ℹ️
สิ่งที่ต้องมีก่อนเริ่ม
✅ บัญชี Google (Gmail)
✅ เบราว์เซอร์ Chrome / Edge
✅ ไฟล์ IT-ServiceDesk-v5.html (ดาวน์โหลดแล้ว)
1
เปิด Google Sheet (สร้างให้แล้ว ✓)
Sheet "IT ServiceDesk DB" ถูกสร้างไว้ให้แล้ว คลิกเพื่อเปิด
💡
วิธี: กด + Blank Spreadsheet → ตั้งชื่อว่า "IT ServiceDesk DB" → จำ URL ไว้
2
เพิ่ม Apps Script
ใน Google Sheets ที่สร้าง → เมนู Extensions → Apps Script
ลบ code เดิมทั้งหมด แล้ว วาง code ด้านล่าง → กด Save (Ctrl+S)

        

        

          
          
        

      

    


    
    

      

        
3

        

      

      

        
Deploy เป็น Web App + รัน Setup

        
ทำ 2 อย่างในหน้า Apps Script:

        

          

            ①
            
Run Setup: เลือก function initSheets → กด ▶ Run → อนุมัติ Permission → รอจนเสร็จ

          

          

            ②
            
Deploy: Deploy → New Deployment → ⚙️ Web App

            • Execute as: Me

            • Who has access: Anyone

            → กด Deploy → Copy Web App URL
